# Samsung Mobile Market Performance Analysis (2019-2024)

## Project Overview
This project involves a comprehensive analysis of Samsung Mobile's financial and market performance across global regions. Using SQL for data extraction and transformation, I analyzed a dataset covering Q1 2019 to Q4 2024 to identify trends in 5G adoption, revenue growth, and product market share.

## Key Insights
* **Revenue Leadership:** Identified Latin America as the highest-grossing region with $173M in revenue for the analyzed period.
* **Product Performance:** Discovered that while the **Galaxy A14 5G** dominates market share (16.7%), the **Galaxy Z Fold3 5G** is a top revenue generator ($80.9M).
* **5G Expansion:** Tracked the growth of 1.8 billion 5G subscribers, with Europe leading in network coverage at 25.1%.

## Technologies Used
* **SQL:** Data cleaning, aggregations, and multi-table joins to calculate KPIs.
* **Data Visualization:** Created interactive dashboards to represent regional performance and product trends.

## Data Points Analyzed
* Total Revenue and Units Sold.
* 5G Subscriber growth and Average 5G Speeds (Mbps).
* Regional market share (Latin America, North America, EMEA, Asia-Pacific, Europe).
* Product-level performance for the Galaxy S, Note, and Z-Series.

## How to Use This Repository
1.  Download the `Samsung_Analysis.sql` file.
2.  Run the script in your preferred SQL environment (MySQL, PostgreSQL, or SQL Server).
3.  The queries are organized to first clean the data and then generate the specific metrics used in the attached visualizations.
